# Substack Reading Digest Analysis

## Project Overview
This mini project analyzes Substack email reading habits using Google Sheets.

The goal is to understand which content categories receive the most reading time.

## Dataset
The dataset includes:

- Date
- Sender
- Subject
- Category
- Read Time (minutes)

## Analysis Steps
- Cleaned and organized data in Google Sheets
- Created pivot tables to summarize reading time by category
- Built a bar chart to visualize category trends

## Key Findings
- The **"down"** category had the highest total read time
- The **"weather"** category had the lowest

## Tools Used
- Google Sheets
- Pivot Tables
- Charts
- GitHub for project documentation

## Files Included
- `substack_emails.csv` — raw dataset
- `pivot_read_time_by_category.csv` — pivot table summary
